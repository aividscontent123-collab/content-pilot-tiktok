# Weekly Engine Checklist

## Monday scan

- Refresh YouTube and Instagram stats.
- Update Published Library.
- Append Stats Snapshots.
- Review pipeline statuses.
- Check scheduled queue and backups.
- Check automation health.
- Check error log.

## Plan output

For each brand, produce:

- recommended batch size
- topic candidates
- title candidates
- hashtag plan
- publishing slot plan
- known blockers

## FSR batch creation

- Create 3 to 5 original story concepts.
- Confirm titles before rendering.
- Generate scripts in Brian pacing.
- Generate Brian voiceovers.
- Render from cropped.mp4 with fixed overlay.
- Upload Drive Review links.
- Set Pipeline to READY_FOR_REVIEW.

## Schedule output after approval

- Set Pipeline to SCHEDULED.
- Add queue rows.
- Create scheduled backup package.
- Set automation or queue runner.

## Publish output

- Publish to correct platform accounts only.
- Verify platform URLs.
- Verify Instagram hashtags.
- Update Published Library.
- Update Publishing Log.
- Update Pipeline to PUBLISHED.
